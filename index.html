GSWAM: A Unified Gaussian World Action Model for Robotic Manipulation

Anonymous Authors
Paper arXiv Code

Abstract

World action models provide a promising learning paradigm for robotic manipulation by jointly modeling future observations and robot actions. However, most existing methods operate in 2D image spaces, making it difficult to explicitly capture spatial scene physical dynamics, maintain cross-view geometric consistency, and effectively align visual observations with robot actions. To address this, we propose GSWAM, a world action model based on 3D Gaussian Splatting, which jointly models scene dynamics and robot actions in a unified 3D space. We first compress 3DGS scenes into geometrically consistent latent representations using GSVAE, which is regularized by cross-view correspondence constraints. Furthermore, we align scene and action tokens into the shared anchored space, enabling GSWAM to explicitly model the joint distribution between 3D scene dynamics and robot actions. Based on this representation, we design a 3D-anchor-based spatial diffusion transformer and introduce a relative attention mechanism to model spatial interactions between scenes and actions. We further employ decoupled flow timesteps for the scene and action modalities, allowing the model to jointly learn scene-action generation and action-conditioned future scene prediction. Experiments on both simulated and real-world robotic manipulation tasks demonstrate that GSWAM outperforms representative existing methods in both action generation and future scene prediction.
